WordPressin asentaminen

Tämä ei ole helppo prosessi! Jos sinulla ei ole kokemusta ohjelmien asentamisesta palvelimelle, tämä prosessi ei luultavasti ole sinua varten - tahdot ehkä mielummin mennä WordPress-sivuille ja luoda blogi-käyttäjätunnuksen ilmaiseksi. Jos kuitenkin tahdot säätää enemmänkin WordPressin asetuksia, voit seurata tätä prosessia.

Vaatimukset 

Tarvitset järjestelmänvalvojan perustaidot ja seuraavat ohjelmat asentaaksesi WordPressin palvelimellesi. On tämän käyttöoppaan kattavuuden ulkopuolella esitellä jokainen teknologia, joten oletamme että ennakkotiedot ovat hyvät.

  • MySQL 4 tai parempi (muut SQL -versiot vaativat lisäasetuksia, joita ei käsitellä tässä kappaleessa).
  • PHP 4 tai parempi.

Vaihtoehtoisesti htaccess -pääsy tai server rewrites (tarvitaan pysyvien kustomoitujen permalinks -linkkien tekemiseen).

Tarvitset myös seuraavat taidot.

    • Tunnet FTP/FTPS -tiedostonsiirtoprotokollan sekä hakemistojen oikeuksien asettamisen.
    • TAI olet taitava purkamaan arkistoja ja asettamaan tiedostojen omistusoikeuksia päätteellä

 *Verkossa toimivan tiedostonhallintaohjelman käyttö on erittäin suositeltavaa, koska muutamia tiedostoja täytyy ladata.

WordPressin asentaminen FTP/sFTP -tiedostonsiirrolla

 

  1. Lataa WordPress.
    http://wordpress.org/latest.zip
  2. Tuplaklikkaa purkaaksesi "latest.zip" .tiedoston. Tämä luo "wordpress" -hakemiston.
  3. Luo uusi hakemisto nimellä "uploads". Se tulee "wp-content" -hakemiston sisälle "wordpress" -hakemistossa. 
  4. Lataa "wordpress" -hakemiston sisältö verkkosivusi juureen tai alihakemistoon, esimerkiksi "blog" -hakemistoon.
  5. Selvitä missä voit asentaa tiedostojen oikeudet tai käytä käskyä CHMOD FTP/FTPS -ohjelmassasi.
  6. Aseta juurihakemiston ja "uploads" -hakemiston käyttöoikeudet muotoon read/write/execute (777) kaikille käyttäjille.

 

WordPressin asentaminen Fantastico Script Installerilla

Monet verkkosivut on varustettu cPanelilla. cPanelissa on automaattinen asennusohjelma, jota kutsutaan nimellä Fantastico Script Installer. Yksi ominaisuus siinä on WordPressin automaattinen asentaminen. Tehdäksesi tämän navigoi Fantastico Script Installeriin cPanel-ohjauspaneelista ja seuraa ohjeita asentaaksesi WordPressin.

Installing WordPress via the command line

 

  1. Lataa WordPressin uusin versio. 
    curl -O http://wordpress.org/latest.tar.gz
  2. Pura se hakemistoon "./wordpress/". 
    tar zxf latest.tar.gz
  3. Siirrä sisältö verkkopalvelimesi juurihakemistoon tai siellä olevaan alihakemistoon. 
    mv wordpress/* webroot/
  4. Tee lataushakemisto sinne, missä WordPress sen odottaa olevan. 
    mkdir -p webroot/wp-content/uploads
  5. Etsi käyttäjä, jonka alla Apache toimii, tässä tapauksessa "www". 
    grep -r ^User\  /etc/httpd/
  6. Aseta juurihakemiston ja lataushakemiston käyttöoikeudet. 
    sudo chown www webroot/ webroot/wp-content/uploads/

 

Asennuksen viimeisteleminen

Ennen kuin jatkat, tarvitset:

  • Käyttämäsi tietokannan nimen, esimerkiksi: user1db
  • Käyttäjänimen, jolla pääsee tuohon tietokantaan, esimerkiksi: user1
  • Käyttäjänimen salasanan.
  • Palvelimen, jolla SQL-tietokantasi on (jos käytät tätä omalta palvelimeltasi, se on luultavasti localhost, mutta muuten sinun tulisi saada se selville verkkosi ylläpitäjältä).

 

Mene selaimellasi verkkosivusi etusivulle, esimerkiksi: http://mywordpress.com/

Automaattiset asetusohjelmat opastavat sinut prosessin seuraavan osan läpi.

  1. Klikkaa linkkiä, joka sanoo: "Create a wp-config.php file through a web interface".
  2. Klikkaa linkkiä, joka sanoo: "If you"re all ready, let's go!".
  3. Kirjoita vaaditut tiedot, jotka olet jo kerännyt. Taulukkojen etuliikettä varten muuta tarjottu "wp_" johonkin merkitykselliseen, kuten "blog1_".
  4. Klikkaa "Submit".
  5. Jos virheitä ei tapahdu, klikkaa linkkiä "If you are ready, time now to run the install!".
  6. Kirjoita ensimmäiseen kenttään blogisi nimi. Huomaa: tämä voidaan vaihtaa milloin tahansa.
      Esimerkki: "WordPressin testaaminen".
  7. Kirjoita sähköpostiosoitteesi. Tämä on ainoa tapa saada järjestelmänvalvojan salasana, mikäli hukkaat sen ja WordPressissä ei ole muita järjestelmänvalvojia.
  8. Paina nappia, jonka nimi on "Install Wordpress".

Jos joudut lopettamaan työskentelyn tähän, varmista että olet joko:

  • Saanut järjestelmänvalvojan salasanan sähköpostilla tai
  • Kirjoittanyt järjestelmänvalvojan salasanan turvalliseen paikkaan.
Jos et ole tehnyt näin, joudut poistamaan tietokannan taulukon ja aloittamaan alusta, mitä ei käsitellä tässä luvussa. Muussa tapauksessa, kopioi järjestelmänvalvojan salasana ja paina "Login" -nappia.

 

Järjestelmänvalvojan ensimmäinen salasana on "admin" ilman lainausmerkkejä. Jos työskentelet jaetulla tietokoneella, älä laita rastia "Remember me" -laatikkoon.

Ensimmäinen asia, joka tulisi tehdä sisäänkirjauduttaessa, on uuden käyttäjän luominen itsellesi muistettavalla salasanalla.

  1. Klikkaa Users -välilehteä ylhäällä olevan valikkopalkin oikealla puolella.
  2. Vieritä sivua alaspäin, kunnes näet otsikon "Add New User".
  3. Kirjoita käyttäjänimesi, sähköpostiosoitteesi ja uusi salasanasi. Aseta Role -alasvetovalikkoon Administrator ja paina "Add User" -nappia.

 

Lopuksi sinun tulisi muuttaa verkon juurihakemiston käyttöoikeudet takaisin joksikin turvallisemmaksi. Muuta juurihakemiston käyttöoikeuksia tai käytä CHMOD-komentoa. Aseta oikeuksiksi read/write/execute vain omistajalle, ja read/execute ryhmille ja muille käyttäjille. Komentoriviltä voit käyttää komentoa:

sudo chown `whoami` webroot/

 

Asennus ja jotkin vähäisemmät asetukset on nyt tehty. Jos selaat blogisi etusivulle, sinun pitäisi nyt nähdä WordPressin perusasennus, jossa ei ole juuri mitään sisältöä, ja blogisi otsikko näkyy sivun yläosassa.
