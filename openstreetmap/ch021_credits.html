Subjects: collective mapping, open data, open map

Summary: This book provides an introduction to collaboratively editing maps with OpenStreetMap and sharing the maps with others. It features stories about communities using the OpenStreetMap data and describes the tools used for map editing, including paper based data collection, mobile map applications, and serving map data using a web server.
