<html dir="None"><body><div id="main-content"><h1>Editing with mobile editors
</h1>
<p>There are several editors available for mobile phones. Mapzen POI Collector is an application available for both Android and iPhones. Mapzen is aimed to be easy to use and make it easy for ordinary people to start mapping easily. </p><p>Vespucci is an application for Android phones that can be used for GPS point gathering and basic OpenStreetMap editing. It is aimed at more advanced users, as you need to know the tagging scheme used.</p><h2>Editing with Mapzen POI Collector</h2><p>Mapzen is available from both the Android Marketplace (<a href="https://market.android.com/details?id=com.mapzen" target="" title="">https://market.android.com/details?id=com.mapzen</a>) and the iTunes Store (<a href="http://itunes.apple.com/gb/app/mapzen-poi-collector/id338079717?mt=8" target="" title="">http://itunes.apple.com/gb/app/mapzen-poi-collector/id338079717?mt=8</a>). When you enter an area that you want to you open the app and the map will open in the location that the map was last shown. To get to your current location you select the android menu, and choose <strong>My location</strong>. The screenshots here are taken from the Android version of the App. The iPhone version is similar.</p><p>Mapzen is focused on simple editing of points of interest (POIs). The current set of POIs will automatically load. You can select a POI to get more info about it by tapping the little blue arrow.</p><h1><img src="_booki/openstreetmap/static/mapzen-2.png" alt="" align="" width="540" height="298" class=""><span class="Apple-style-span"> </span></h1><p>Editing a current point involves selecting a point and going through the menu system to change the properties of it, and then finally saving it.</p><p><img src="_booki/openstreetmap/static/mapzen-edit-2.png" width="100%"> </p>
<p>We did not go very in depth into Mapzen POI Collector, because the application is intended to be intuitive. </p><h2>Editing with mobile editor Vespucci</h2>
<p>Vespucci is an advanced editor that allows you to update the OSM data in the field. Refer back to the JOSM section to understand tags. You also need to know about the OSM data model (a chapter in this book) and the tagging scheme which is described on the OSM wiki on the Map Features page: <a href="http://wiki.osm.org/wiki/Map_Features" target="" title="">http://wiki.osm.org/wiki/Map_Features</a>. You currently can't edit relations with Vespucci.</p><h3>Installing Vespucci on Android</h3>
<p>You can download Vespucci from the Android Market by searching for "Vespucci" or going to the following page (<a href="https://market.android.com/details?id=de.blau.android" target="" title="">https://market.android.com/details?id=de.blau.android</a>), and following the usual installation procedure for your phone.</p>
<h3>Setting up Vespucci</h3>
<p>To be able to edit OSM data you need to enter your OSM username and password into the settings.</p>
<p><img src="_booki/openstreetmap/static/Vespucci%20preferences%20osm%20username-password.png" alt="Screenshots showing how to enter your username and password info into Vespucci" align="" width="100%" class=""></p>
<ol><li>Start Vespucci application on your Android.</li>
  <li>Press the <strong>Menu </strong>button on your Android phone.</li>
  <li>Select <strong>More </strong>and <strong>Preferences</strong>.</li>
  <li>Enter your <strong>OSM Username </strong>and <strong>OSM Password</strong>.</li>
</ol><h3>Downloading OSM Data</h3>
<p>Once you are in an area you need to start the app, get your current location and download the current OSM data for the area.</p>
<ol><li>To get your current location to show press the phone's menu button, <strong>More</strong>, <strong>GPS...</strong> and <strong>Follow GPS position</strong> or <strong>Start/pause GPS track</strong>.</li><li><span class="Apple-style-span">Vespucci should center the map on your current location.</span></li><li><span class="Apple-style-span"></span>To download the current data press the phone's menu button, and select <strong>More</strong> and then <strong>Transfer...</strong> and <strong>Download current view</strong>.<img src="_booki/openstreetmap/static/VespucciOverview.png"></li>
  </ol><p><em>Screenshot showing a GPS track with the underlying OSM data.</em></p>
<h3>Moving the map</h3>
<p>You can move the map around when you are in move mode by dragging the map around. There are zoom in and out buttons that allow you to change the zoom level. You can also use the pinch gesture to zoom in and out of the map.</p>
<h3>Moving nodes
</h3>
<ol><li>Push the Android <strong>Menu </strong>button and select <strong>Edit</strong>. <img src="_booki/openstreetmap/static/icons.png"></li>
  <li>Nodes and ways are now highlighted.</li>
  <li>You can move the nodes around by pressing a node to select it and then pulling it with your finger.</li>
</ol><h3>Editing tags
</h3>
<ol><li>Push <strong>Menu </strong>button on your Android. Select <strong>Edit Tags</strong>.  </li>
  <li>Touch on a node.</li>
  <li>You will now see a menu of nodes.</li>
  <li>Select the node you want to edit.</li>
  <li>You can now edit the <strong>Key </strong>and <strong>Value </strong>pairs of the node.</li>
  <li>Enter a new <strong>Key</strong> and <strong>Value </strong>pair in the empty slots to add a new <strong>Key/Value</strong>.</li>
  <li>When you are finished editing the values you can push <strong>Okay</strong>.</li>
</ol><h3>Creating new features</h3>
<ol><li>Press the <strong>Menu </strong>button and select <strong>New</strong>.<img src="_booki/openstreetmap/static/CreateMode.png"></li>
  <li>Pushing once on the map creates a point. You can then switch to edit tags mode and re-select that node to give it some properties</li>
  <li>Pushing the map many times creates a way.</li>
  <li>Ending a way by pushing on the starting point creates an area.</li>
</ol><h3>Appending ways
</h3><ol><li>Press <strong>Menu </strong>button.</li>
  <li>Select <strong>Append</strong>.</li>
  <li><span class="Apple-style-span">You are now able to continue an old way by adding new points to the way.  </span></li>
</ol><h3>Deleting Nodes</h3>
<ol><li>Press the <strong>Menu </strong>button.</li>
  <li>Select <strong>More</strong>.</li>
  <li>Select <strong>Erase node</strong>.</li>
  <li>Push on a node to delete it.</li>
</ol><h3>Saving your changes</h3><p>Once you have made some changes to the data in the area you will want to share those with the rest of the OSM community.</p>
<ol><li>Press the <strong>Menu</strong> button on your Android.</li>
  <li>Select <strong>More</strong>, and then select <strong>Transfer</strong>.</li>
  <li>Select <strong>Upload data to OSM server</strong>. You will be shown a list of the changes that you are about to upload and make live on the OSM server, and be asked to enter a comment about what the changes are. You then tap the <strong>upload</strong> button.</li>
