<div id="main-content"><h1>Viivatyylit
</h1>
<h2><span id="result_box" class="long_text"><span></span></span>
</h2><span id="result_box" class="long_text"><span>
<br></span><span>Rajaviivojen ja geometristen muotojen tyyliä voidaan muuttaa, jotta vältetään virheiden ilmestyminen viivojen leikkauskohtiin. Tähän tarkoitukseen käytetyt komennot esitellään alla. </span><span></span></span>
<h2>smooth
  <br></h2>
<p>Metodi <code>smooth()</code> mahdollistaa reunojen tasoittamisen. Sen avulla vältetään portaikkoefekti, joka tulee näkyville diagonaalisissa viivoissa.
  <br></p>
<pre>line(10, 0, 100, 90);  // Ilman tasoitusta

//Aktivoimme tasoituksen
smooth();
line(0, 10, 90, 100); // <span id="result_box" class="long_text"><span>Viiva tasoitettu
</span></span></pre> <img src="_booki/processing/static/smooth-fi_1.png"><h2>strokeWeight
</h2>
<p> Metodi <code>strokeWeight()</code> mahdollistaa viivan paksuuden muuttamisen.
  <br></p>
<pre>line(10, 0, 100, 90); // Yhden pikselin paksuinen viiva
strokeWeight(5); // Asetetaan paksuudeksi viisi pikseliä
line(0, 10, 90, 100); // Viiden pikselin paksuinen viiva
</pre>
<p> <img title="strokeWeight" alt="strokeWeight" src="_booki/processing/static/strokeWeight-fi.png" height="100" width="100"></p>
<h2>strokeCap
</h2>
<p><span id="result_box" class="long_text"><span>Metodi <code>strokeCap ()</code> muuttaa viivan lopun ulkoasua. Tämä metodi ei ole hyödyllinen muodoissa. Sen arvot voivat olla </span><span>SQUARE, PROJECT tai ROUND. </span><span>Oletusarvoisesti käytetään arvoa ROUND. </span><span>Tämä metodi ei toimi OpenGL tai P3D -tilassa.</span></span>
</p>
<pre>strokeWeight(10); // Määrittelemme paksuudeksi 10 pikseliä
strokeCap(ROUND);
line(20, 40, 60, 80);

strokeCap(PROJECT);
line(20, 20, 80, 80);

strokeCap(SQUARE);
line(40, 20, 80, 60);
</pre>
<p><img title="strokeCap" alt="strokeCap" src="_booki/processing/static/strokeCap-fi.png" height="100" width="100"><br></p>
<h2>strokeJoin
</h2>
<p><span id="result_box" class="long_text"><span>Metodi <code>strokeJoin()</code> muuttaa kulmien ulkoasua. Sillä voi olla arvot</span><span> MITER, BEVEL tai ROUND. </span><span>Oletusarvoinen tila on MITER. Tämä metodi ei toimi </span><span>OpenGL tai P3D -tiloissa.</span></span>
</p>
<pre>size(300, 100); // Muuttaa kuvan kokoa

strokeWeight(10); // Määritellään paksuus 10 pikseliä

strokeJoin(MITER); // Suorakulmat
rect(20, 20, 60, 60);

strokeJoin(BEVEL); // Karkeat kulmat
rect(120, 20, 60, 60);

strokeJoin(ROUND); // Pyöristetyt kulmat
rect(220, 20, 60, 60);
</pre>
<p>
  <br><img title="strokeJoin" alt="strokeJoin" src="_booki/processing/static/strokeJoin-fi.png" height="100" width="300"></p>
